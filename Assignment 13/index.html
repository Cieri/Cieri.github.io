Style Guide

Responsive Design


All pages should utilize responsive design via the Bootstrap grid. Views should be styled for the small, medium, and large breakpoints.

File Management and Naming Conventions

Site Map

A site map should be included and accessible from a link within the footer.

Folder Naming

Folder names should be a single, easy to recognize words in all lowercase letters – such as images, scripts, etc.

File Naming

File names should be easy to recognize words, separated by a period, in all lowercase letters – such as company.logo.jpg, app.component.ts, etc.

Buttons

Button options are available in light and dark, as well as small and large, and can be adjusted according to the purpose. The buttons should also have an inverted color display on hover and focus. Buttons should have a border radius of 25pts to maintain consistency throughout any webpages or landing pages.

Still Graphics and Photographs

Images should be at least 72 pixels per inch (PPI). Using CSS, the images should be set to display at the following sizes at the following breakpoints:

  • Large devices (minimum width: 992 pixels): 500 pixels wide
  • Medium devices (minimum width: 768 pixels): 350 pixels wide
  • Small devices (minimum width 576 pixels): 250 pixels wide
  • Extra small devices (maximum width 575.99 pixels): 200 pixels wide

Images should be displayed in a circlular clipping mask, with the exception of the hero image. Drop shadows and borders, as well as overlay with or without copy, may be used where appropriate, but one should refrain from using any filters other than greyscale.

Responsive Media Query Examples:

img {
width: 500px;
}

@media only screen and (max-width: 576px) {
img {
width: 200px;
}
}

@media only screen and (max-width: 768px) {
img {
width: 250px;
}
}

@media only screen and (max-width: 992px) {
img {
width: 350px;
}
}

Audio, Video, and Animation


Audio files should be mp3 and may not be set to autoplay.

Videos should either be embedded Youtube or Vimeo videos and must be responsive to the width of the user's device. Videos may not be set to autoplay.
